# **App Name**: CubaRenta

## Core Features:

- Listado de Autos: Mostrar una lista de autos disponibles con fotos, descripciones y precio por día.
- Selección de Fechas: Permitir al usuario seleccionar fechas de inicio y fin para la renta.
- Cálculo de Precio: Calcular el precio total de la renta basado en el número de días y el precio por día del auto seleccionado.
- Guardar selección: Permitir al usuario guardar su selección para consultar luego.
- Recomendaciones de Autos con IA: Ofrecer sugerencias de vehículos basados en las preferencias ingresadas (por ejemplo, "autos económicos", "autos para familias"). LLM uses reasoning to act as a tool which makes these recommendations.
- Confirmación de Renta: Mostrar una página de confirmación con el resumen de la renta y los detalles del cliente.

## Style Guidelines:

- Color primario: Azul profundo (#1A5276) para reflejar confianza y profesionalismo.
- Color de fondo: Gris claro (#F2F4F4) para una apariencia limpia y moderna.
- Color de acento: Naranja (#E67E22) para resaltar botones de acción y elementos importantes.
- Fuente para encabezados: 'Poppins', una sans-serif geométrica que transmite modernidad y precisión.
- Fuente para cuerpo del texto: 'PT Sans', una sans-serif humanista que facilita la lectura en párrafos largos.
- Utilizar iconos claros y minimalistas para representar las diferentes categorías de autos y funciones de la app.
- Diseño limpio y espacioso, con márgenes generosos y una jerarquía visual clara para facilitar la navegación.
- Animaciones sutiles al cargar nuevos autos o al realizar una selección, para mejorar la experiencia del usuario.
